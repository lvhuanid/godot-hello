# godot-hello

A Godot 4 project with a bouncing ball and a player-controlled character.

## Features

- A ball that bounces off walls with realistic reflection (using collision normals).
- A player character (cyan square) controlled by arrow keys / WASD.
- Walls around the play area.

## Controls

- Move player: Arrow keys (or WASD if mapped in Godot Input Map).

## Run

Open the project in Godot 4.7+ and press F5 to run `main.tscn`.
